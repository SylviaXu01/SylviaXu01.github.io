1. NLP简介

1.1 NLP概述

NLP：自然语言处理
使计算机理解，解释，生成人类语言，并基于这些能力对文本数据进行决策

NLP = NLU（自然语言理解 输入语言，输出结构化信息）
+ NLG（自然语言生成 输入结构化信息，输出语言）

处理过程层次：
词法分析 -> 句法分析 -> 语义分析 -> 语用分析

主要挑战：
语言的歧义性（词法 结构）
常识与世界知识
深层推理能力
语境与文化
模型幻觉
数据质量与稀缺性
低资源语言
数据偏差（训练数据的偏见）
计算成本
可解释性与安全性
